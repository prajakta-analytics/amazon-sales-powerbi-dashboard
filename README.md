# Amazon Sales Analysis Dashboard (Power BI)

## Overview
Interactive Power BI dashboard analyzing Amazon product sales performance, category trends, and top products.

## Tools Used
Power BI, Power Query, DAX, Data Modeling

## Key Insights
- YTD & QTD sales tracking
- Category-wise performance
- Top products by sales & reviews
